## ðŸš€ Overview

**Document Classifier** is a complete end-to-end automation system designed to classify and organize unstructured documents into a clean, standardized folder structure.

It allows users to:

- Drop unorganized documents into a source directory  
- Automatically classify documents into predefined categories  
- Generate structured folder hierarchies  
- Organize files using rule-based or AI-assisted logic  
- Produce JSON mappings and summary reports for traceability  

This project is ideal for **corporates, legal firms, hospitals, researchers, data teams, and enterprises** managing large document repositories.
